# PizzaMania

Welcome to PizzaMania! This project is a WordPress-based website for a fictional pizza shop, showcasing their menu, services, and more.

## Description

PizzaMania is a WordPress project aimed at demonstrating my skills in web development, specifically using WordPress as the content management system. It features a custom theme developed from scratch to suit the needs of a pizza shop, including customizable pizza options, menu listings, contact information, and more.

## Features

- Customizable pizza options with 'Your Signature Pizza'
- Menu listings for various pizza flavors and side dishes
- Contact form for customer inquiries and feedback
- Responsive design for seamless browsing on all devices
- Developed using WordPress, PHP, HTML, CSS, and JavaScript

## Installation

To run this project locally, follow these steps:
1. Clone the repository: `git clone https://github.com/vineeth0306/wordpress-tut.git`
2. Set up a local development environment with WordPress installed
3. Move the cloned repository into the `wp-content/themes/` directory of your WordPress installation
4. Activate the 'PizzaMania' theme in the WordPress admin panel
